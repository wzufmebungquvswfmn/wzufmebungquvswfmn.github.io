# 博客构建

## mdbook
当前博客采用mdbook构建，这是一个rust编写的markdown文档构建工具，可以将markdown文档转换为html文档，同时支持目录、搜索、高亮等功能。同时它也是rust官方文档的构建工具。

## 使用方法
cargo install mdbook
mdbook init
mdbook serve / mdbook build

## 目录隐藏
注册在SUMMARY.md的所有章节都会在侧边栏显示，如果想隐藏某些章节，可以在 `src/theme/custom.css` 中添加如下样式：
```css
#mdbook-sidebar li.chapter-item:has(> span.chapter-link-wrapper > a[href$="blog/博客构建.html"]),
body.sidebar-iframe-inner li.chapter-item:has(> span.chapter-link-wrapper > a[href$="blog/博客构建.html"]) {
  display: none !important;
}

```
