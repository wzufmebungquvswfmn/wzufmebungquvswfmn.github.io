# 集合数据结构
Rust 标准库文档的 `std::collections` 模块中这样写到：  

`you should probably just use Vec or HashMap. These two collections cover most use cases for generic data storage and processing.`  

也就是说绝大多数场景里，`Vec` 和 `HashMap` 已经能够解决问题。但无论如何，Rust 标准库还是提供了多种集合数据结构来满足不同的需求：  

- [Vec](集合数据结构/vec.md)  
- [HashMap/HashSet](集合数据结构/hash.md)  
- [BinaryHeap](集合数据结构/binaryheap.md)  
- [VecDeque](集合数据结构/vecdeque.md)  
- [BTreeMap/BTreeSet](集合数据结构/btreemap.md)  

# 如何选择集合

`Vec`、`HashMap`、`HashSet` 自不必多说。  

双向队列有时也会有用，尤其是 BFS 中从队头取元素、从队尾加入元素，使用 `VecDeque`。

如果需要每次取出最大值或最小值，使用 `BinaryHeap` 作为优先队列，它提供了性能极好的有序插入。

如果需要 key 保持有序，或者需要范围查询，使用 `BTreeMap` / `BTreeSet`。
