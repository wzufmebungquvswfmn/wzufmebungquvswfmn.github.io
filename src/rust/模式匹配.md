# 这里记录模式匹配的一些零碎知识点
- [全模式列表](模式匹配\全模式列表.md)
## 常用模式匹配工具
match, if let, while let, match!宏, let else等

## 不可反驳模式
let, match, fn, for等要求完全匹配覆盖，被称作`不可反驳模式`（irrefutable patterns）

## 可反驳模式
if let, while let, let else等允许不完全匹配，被称作`可反驳模式`（refutable patterns）

- let else 的一个显著特点是其创建的变量具有更广泛的作用域，而非if let那样仅在特定分支内使用
