#  泛型(Generic) 和特征 (Trait)

记录一些零碎的泛型和特征的知识点

## 泛 型
泛型是一种类型参数，可以是任何类型，在编译时会被替换为具体的类型
-  fn func<T>(arg: T) -> N ;
-  struct S<T> { arg: T, .. };
-  enum E<T> { A(T), B, .. };
-  impl<T: `约束条件`> S<T> { fn func<T>(arg: T) -> N { ..`T受约束的调用`.. } }
    -  `约束条件` 为 `特征` 或 `类型` 或 `特征+类型` 或 `特征+特征` 或 `特征+特征+类型` 等

## 特征
-  trait TraitName { fn func(&self) -> N; }
-  impl TraitName for S { fn func(&self) -> N { .. } }
-  fn func( arg: impl TraitName ) <==> fn func<T: TraitName>( arg: T )
-  fn func( arg: impl TraitName + TraitName2 ) <==> fn func<T: TraitName + TraitName2>( arg: T ) //多重约束
